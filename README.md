# Caesar Cipher 

Ruby solution for the Caesar Cipher exercise, from The Odin Project.

## Description
This project implements a Caesar Cipher method of switching letters in a string by a given number, while leaving unchaged all non-letter characters, so the code returns the following answer:
caesar_cipher("What a string!", 5)
"Bmfy f xywnsl!"

## What I practiced
- Ruby methods
- Strings and characters
- Conditionals
- Character conversion using map, ord and chr
